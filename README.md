# F1 Monaco 2025 Analysis

1. **Lap Delta** : Écart de temps par rapport au pole position.
2. **Average Speed Comparison** : Comparaison de la vitesse moyenne.
3. **Qualification Spread** : Répartition des écarts par pilote.
